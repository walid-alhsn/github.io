Portfolio remake in plain HTML, CSS and JavaScript.

Files:
- index.html
- styles.css
- script.js

How to use:
1. Open index.html in your browser.
2. Edit the text, links, and project details to match your real portfolio.
3. Replace placeholder project images or gradient blocks if you want actual screenshots.
4. For GitHub Pages, upload these files to a repository and enable Pages.

Important:
- The contact form currently uses front-end validation only.
- If you need it for your assignment, connect it to Supabase so submissions are stored in the database, as required by the brief.
